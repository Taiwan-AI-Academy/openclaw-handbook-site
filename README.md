# AIA蝦蝦 🦐 — 台灣人工智慧學校 AI 助手應用案例

AIA蝦蝦是台灣人工智慧學校（AIA）基於 [OpenClaw](https://github.com/openclaw/openclaw) 部署的 AI 助手，透過 Google Chat、LINE 等多通道與同仁協作，落實 AI 融入日常行政與業務的實踐。

本站整理了 AIA 團隊實際使用蝦蝦的案例，展示 AI 助手如何提升組織效率。

---

## 📋 使用案例總覽

### 🔧 Google Workspace 整合
- **Gmail 郵件讀取** — 讀取、摘要每日信件，快速掌握重點
- **Google Calendar 行事曆提醒** — 每日自動推送當天行程
- **Google Sheets 試算表操作** — 讀取與寫入試算表資料（如 ISO 資產盤點 CIA 評分）
- **Google Docs 文件建立** — 依需求自動建立文件並存入指定雲端資料夾
- **Google Forms 表單建立** — 透過對話設計問卷，自動建立 Google 表單
- **OAuth 多帳號授權管理** — 各同仁獨立授權，帳號安全分流

### 🤖 自動化排程
- **輿情監控** — 每日自動搜尋 AI 相關新聞並推送摘要
- **行事曆每日提醒** — 定時推送當日與近期行程
- **OpenClaw 版本檢查** — 每日檢查是否有新版本
- **Google Meet 會議記錄自動化** — 自動偵測新會議逐字稿，生成結構化會議記錄存入 Google Docs
- **笑話排程** — 定時推送笑話活絡氣氛

### 💬 多通道溝通
- **Google Chat** — 主要工作溝通管道，支援多人群組與 1:1 對話
- **LINE** — 支援語音訊息、群組互動、五子棋遊戲
- **LINE 語音訊息** — TTS 合成語音並以 M4A 格式發送

### 📝 內容生成與文件處理
- **教案表單設計** — 根據課程大綱快速產出結構化教案
- **會議記錄生成** — 從逐字稿萃取決議與待辦事項
- **信件撰寫與審閱** — 草擬、檢視、建議修改信件內容
- **LINE / 社群訊息精簡** — 將長信件轉為適合通訊軟體的精簡版本
- **ISO 27001 資產盤點** — 讀取 SOP 與評等標準，自動填寫 CIA 評分與理由

### 🎮 娛樂與生活
- **五子棋遊戲** — LINE 上可玩的文字版五子棋
- **天氣查詢** — 即時天氣與預報
- **食譜建議** — 根據食材推薦料理
- **生活資訊查詢** — 交通、旅遊住宿資訊整理
