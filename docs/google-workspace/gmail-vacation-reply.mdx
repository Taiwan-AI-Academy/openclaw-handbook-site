## 用對話設定 Gmail 自動回覆

**日期**：2026-02-12
**需求者**：課程同仁
**平台**：Google Chat

### 情境

農曆新年假期前，課程同仁希望設定 Gmail 的休假自動回覆，讓寄信的人知道她正在休假、何時回覆。需要中英文雙語版本。

### 過程

1. 同仁告知休假日期範圍（2/13-2/22）
2. 蝦蝦草擬中英文自動回覆內容
3. 同仁確認後，蝦蝦透過 Gmail API 直接設定：
   - 啟用休假自動回覆
   - 設定開始與結束時間
   - 填入中英文回覆內容
   - 設為所有寄件人都會收到回覆
4. 同仁後來改變主意想取消，蝦蝦也即時透過 API 關閉

### 結果

成功透過 API 設定 Gmail 休假自動回覆，不需要同仁自己進 Gmail 設定頁面操作。設定和取消都只要一句話就完成。

### 使用方式

> 「蝦蝦，幫我設定 Gmail 休假自動回覆，2/13 到 2/22」
> 「蝦蝦，取消我的 Gmail 自動回覆」
> 「蝦蝦，幫我寫一段中英文的休假自動回覆」

:::note
需要先完成 Google OAuth 授權（Gmail 權限），蝦蝦才能操作你的 Gmail 設定。
:::
