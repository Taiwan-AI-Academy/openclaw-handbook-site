# 展覽攤位查詢

**日期**：2026-02-16
**需求者**：團隊成員
**平台**：LINE

## 情境

團隊成員正在國外參加大型 AI 研討會暨博覽會，想要找特定廠商的攤位位置。蝦蝦直接上官方網站查詢，幫忙定位攤位所在展館與編號。

## 過程

1. 收到使用者詢問某國際科技公司的攤位位置
2. 蝦蝦打開博覽會官方網站，進入「展商名單」頁面
3. 在搜尋框搜尋目標廠商名稱
4. 透過分類篩選（國內/國際企業）找到該廠商
5. 取得展館編號（Hall）和攤位編號（Booth）
6. 同時提醒展覽開放時間，避免白跑一趟

## 結果

成功找到目標廠商的攤位位置（展館與攤位編號），並提醒當日展區尚未開放，隔天才正式開展。
