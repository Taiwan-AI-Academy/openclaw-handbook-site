# 國際研討會議程篩選推薦

**日期**：2026-02-15
**需求者**：團隊成員
**平台**：LINE

## 情境

團隊成員即將出國參加一場大型國際 AI 研討會，議程多達數百場分散在不同場館。成員希望蝦蝦幫忙從官方網站篩選出與自己關注領域相關的場次，並按場館和時間排序整理。

## 過程

1. 蝦蝦透過瀏覽器開啟研討會官方網站的 Sessions 頁面
2. 爬取所有場次資料（包含時間、場地、主題、講者、合作夥伴）
3. 根據使用者關注的 5 大領域（AI 教育、人才培育、國家政策、AI Safety、Responsible AI）進行篩選
4. 將符合的場次按時間順序排列，並標記強烈推薦的場次
5. 因 LINE 推送訊息遇到限流，建議使用者改用其他通訊平台取得完整清單

## 結果

- 成功篩選出約 20 場高度相關的 sessions
- 每場包含：時間、場地、主題、領域標籤、講者/機構、推薦等級
- 使用者在出發前獲得完整的議程參考，可據此安排行程
