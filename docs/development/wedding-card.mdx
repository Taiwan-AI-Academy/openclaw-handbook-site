## 互動喜帖網頁製作

**日期**：2026-02-13  
**需求者**：團隊成員  
**平台**：Google Chat

### 情境

團隊成員需要製作一個互動式電子喜帖網頁，希望有精緻的開信封動畫效果，並且能在手機上流暢運作。

### 過程

1. 蝦蝦根據需求設計了完整的單頁 HTML 喜帖，包含三大段落：
   - **信封動畫**：信封從畫面底部飛入 → 翻蓋打開（帶蠟封「囍」）→ 喜帖從信封升起 → 信封淡出消失 → 喜帖 3D 翻轉顯示內文
   - **婚禮資訊區**：莫蘭迪色系現代風格，新郎新娘介紹、婚宴時間地點
   - **RSVP 表單**：出席回覆功能，含親友方別、出席人數
2. 加入花瓣粒子特效（粉色花瓣 + 金色紙屑），在喜帖升起和翻轉時散落
3. 加入喜帖金色光暈脈動效果
4. 經過多次調整：修正信封消失的 CSS 動畫覆蓋問題、調整動畫節奏（總長 7.5 秒）
5. 使用瀏覽器即時預覽截圖確認效果，反覆微調至滿意
6. 部署到內部服務供手機瀏覽測試

### 結果

成功產出一個完整的互動式電子喜帖網頁，純 HTML/CSS/JS 無需框架，包含：
- 流暢的信封開啟動畫序列
- 3D 翻轉喜帖效果
- 花瓣粒子特效
- 響應式設計（手機/桌面皆可）
- RSVP 表單功能
- 向下滾動提示
