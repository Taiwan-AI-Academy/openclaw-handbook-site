## 從 Google Doc 到完整課程網站

**日期**：2026-02-12
**需求者**：工程師 A
**平台**：Google Chat

### 情境

團隊需要為新的 AI 產品經理培訓課程建立一個行銷網站。工程師 A 提供了一份 Google Doc，包含完整的課程資訊（課程介紹、對象、特色、效益、八週課程表、行前準備、報名資訊），希望蝦蝦能直接將內容放進現有的網站模板中。

### 過程

1. 蝦蝦讀取 Google Doc 原文，逐段對應到網站模板的各個區塊
2. 修改 Hero 區塊：將文字標題替換為 SVG logo（全寬顯示）
3. 更新「關於課程」兩段完整保留原文
4. 建立三欄式「課程對象」卡片（現職 PM、部門主管、非 PM 背景）
5. 六格式「課程特色」區塊
6. 七項「課程效益」完整照搬
7. 完整八週課程表（含上下午分課和細項）
8. 新增「學員行前準備」區塊（取代原本的完課證書區塊）
9. 更新報名區的招生對象與課程資訊
10. 移除不適用的場地區塊
11. 用瀏覽器預覽確認效果後，commit 並 push 到 Git

### 結果

一個完整的課程行銷網站，包含八大區塊，設計風格保持一致（深藍+金色主題），所有內容一字不漏從 Google Doc 搬入。從收到需求到完成不到 10 分鐘。

### 使用方式

> 「蝦蝦，幫我用這份 Google Doc 的內容建一個課程網站」
> 「蝦蝦，把這個網站的 Hero 標題換成這張 SVG logo」
> 「蝦蝦，參考 A 網站的設計，幫 B 網站做類似的版面」
