# 每日工作日報

## 自動產生每日工作報告

**日期**：2026-02-12
**需求者**：課程同仁
**平台**：Google Chat

### 情境
團隊成員希望蝦蝦每天自動整理前一天做了哪些事，包含使用量估計，定時發送到工程師群組，方便追蹤 AI 助手的使用情況。

### 過程
1. 設定 cron 排程，每天早上 8:30 執行
2. 讀取前一天的記憶檔取得工作紀錄
3. 查詢 session 狀態取得使用量
4. 整理成條列式報告，自動發送到指定的 Google Chat 群組

### 結果
- 排程成功建立，每天 08:30 自動執行
- 報告包含：工作項目條列 + 使用量估計
- 如果前一天沒有紀錄，會報告「昨日無工作紀錄」

### 使用方式

<ChatDemo platform="gchat">
  <ChatBubble from="user">每天早上 8:30 發工作日報到工程師群組</ChatBubble>
  <ChatBubble from="bot">好的！已設定 cron 排程，每天 08:30 會自動整理前一天的工作紀錄並發送到群組 📋</ChatBubble>
</ChatDemo>

也可以說「給我看昨天的工作紀錄」即時取得報告。
