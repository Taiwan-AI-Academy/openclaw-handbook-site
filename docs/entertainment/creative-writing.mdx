# 創意文字內容生成

**日期**：2026-02-16
**需求者**：團隊成員
**平台**：Google Chat

## 情境

農曆新年除夕，團隊成員在群組拜年聊天，有人要求蝦蝦「畫一張馬年春聯」。雖然在文字頻道無法生成圖片，蝦蝦靈機一動用 ASCII art 排版了一副完整的春聯。

## 過程

1. 收到使用者要求繪製馬年春聯
2. 判斷目前平台不支援圖片上傳
3. 改用文字 ASCII art 排版，模擬春聯版面
4. 撰寫上聯、下聯、橫批，搭配節慶 emoji

## 結果

用純文字排版出一副完整的馬年春聯，包含邊框、上下聯、橫批，搭配 emoji 裝飾，團隊成員覺得有趣。
